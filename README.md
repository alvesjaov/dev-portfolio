
# Portfolio

## Como rodar o projeto localmente

1. Clone o repositório usando a URL do projeto.
2. Acesse a pasta do projeto.
3. Instale as dependências:
	```sh
	npm install
	```
4. Inicie o servidor de desenvolvimento:
	```sh
	npm run dev
	```

## Tecnologias utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
